これは何か

光市の企業・事業者を対象とした、AI（Claude Code）の無料導入支援プロジェクトです。

遠隔で完結します。申込から導入まで最短即日。

自分の事業でAIの導入・推進を進める中で、この技術は隣の事業者にも届けられるし、届けるべきだと考えるようになりました。

一社だけが強くなっても地域は変わりません。意欲のある企業が横につながり、同じ基盤の上で動けるようになれば、光市全体の競争力が上がる。これは私のプロジェクトではなく、参加する全員のプロジェクトです。

小さな事業者も、始めたばかりの個人事業主も、まずは同じスタートラインに立てるようにしたい。そのための導入支援です。

Claude Codeとは

Anthropic社が提供するAIエージェント。会社の経営層・管理職・フリーランスが使うべき技術です。あなたのPC上で動き、自然言語で指示するだけで以下のようなことが実現できます。

// できること

経営 → 経営数字ダッシュボード作成
財務 → 利益率・原価率の可視化
管理 → プロジェクトマネジメントツール作成
労務 → 勤怠・給与計算・申請ツール
取引 → 受注・発注ツール
在庫 → 資材管理・棚卸しツール
集客 → SNS企画・運用パイプライン
調査 → リサーチ自動化
会議 → 議事録作成・関係者通知
Web → ウェブサイト作成
制作 → 自社独自の画像生成

データは社外に保存されません。すべてあなたのPC上で完結します。

FAQ

費用はかかりますか？
導入支援は無料です。Claude Code自体のAPI利用料（月数百円〜数千円程度）は各社ご負担ください。
ITに詳しくなくても大丈夫ですか？
はい。環境に合わせたマニュアルを個別に作成します。
何が必要ですか？
PCとインターネット接続のみ。Windows / Mac どちらもOK。
光市外の企業ですが申し込めますか？
現在は光市内の企業・事業者に限定しています。
データのセキュリティは？
Claude Codeはお手元のPC上で動作します。データは外部サーバーに保存されません。
